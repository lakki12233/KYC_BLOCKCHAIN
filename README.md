**Programming Language:**
- JavaScript (Node.js)

**Libraries:**
- Express.js (for web application framework in Node.js)
- Path (for working with file and directory paths)
- Body-parser (middleware for parsing HTTP request bodies)
- Zlib (for compression and decompression using zlib)
- LZUTF8 (for compression and decompression using LZUTF8)
- Brotli (for compression and decompression using Brotli)
- CryptoJS (for cryptographic operations)
- Nodemailer (for sending emails)

**Tools:**
- Ganache (for Ethereum test network)
- Remix IDE (for compiling smart contract solidity files)

**Web Technologies:**
- HTML
- CSS

**Others:**
- Git (version control system)
- GitHub (for hosting the repository)

This project primarily uses JavaScript for server-side scripting, Node.js for server-side development, and various libraries and tools for compression, encryption, email communication, and blockchain integration.
